# configs

## Key bindings
Dump dconf key bindings
`dconf dump / >~/.config/dconf/keybindings.conf`

Import dconf key bindings
`dconf load / <~/.config/dconf/keybindings.conf`

## Sublime Text
Save config to:
```bash
~/.config/sublime-text-3/Packages/User/Preferences.sublime-settings
```

## Azure CLI
Save config to:
```bash
~/.azure/config
```
